# 🏠 RentEase – Property Rental Management System

RentEase is a full-stack web application designed to help property owners efficiently manage properties, tenants, rental agreements, and payments in one centralized platform.

---
## 🚀 Features

### 🔐 Authentication
- User Registration & Login
- Secure password hashing
- Session-based authentication

### 🏢 Property Management
- Add, update, and manage properties
- Track property status (Vacant / Occupied)

### 👥 Tenant Management
- Add, edit, and delete tenants
- Assign tenants to properties
- Automatic property status update

### 💰 Payment Management
- Record rent payments
- Auto-fill payment details
- Track payment status (Paid / Pending)
- View tenant-wise payment history
- Monthly rent tracking system

### 📄 Agreement Management
- Create rental agreements
- Store rent, deposit, duration, and due date
- Update agreement details

### 📊 Dashboard
- Total properties overview
- Active tenants count
- Monthly revenue insights
- Pending payments tracking

### 📈 Data Analytics (Advanced Feature)
- Data analysis using **Pandas**  
- Insights generation from rental data  
- Trend analysis (monthly payments, occupancy trends)
